# xxl-job

#### 介绍
xxl-job 这个定时任务框架还是非常nice的, 项目中也使用的比较多
基于xxl-job 小改动, 原有项目注解@XxlJob没有删除, 启动后需手动去admin创建任务,  新增了@NewXxlJob注解 ,实现自动创建执行器 与定时任务
